# web2rss

> 通过 AI 将网页列表转为 rss 格式输出

- [x] 支持私有化部署

- [x] 支持 llm

- [x] local router (pull) -> remote router -> save local

- [x] local router (push) -> remote router -> save remote

- [x] fetch (error) -> proxy

## 使用

- 命令行 pip install -f ./requirements.txt

- 修改 config.txt 中的 gpt 参数

- 命令行 python3 ./app.py

- 打开浏览器
